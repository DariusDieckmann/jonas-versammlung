# Erste Schritte

Folgen Sie dieser Schritt-für-Schritt-Anleitung, um Ihre erste Eigentümerversammlung digital durchzuführen.

## 1. Anmelden & Loslegen

### Mit Google oder Microsoft anmelden

1. Klicken Sie auf **"Anmelden"**
2. Wählen Sie **Google** oder **Microsoft**
3. Autorisieren Sie die Anmeldung
4. Sie werden automatisch eingeloggt

➡️ **Sie haben jetzt Zugriff auf das Dashboard**

---

## 2. Organisation erstellen oder beitreten

### Neue Organisation erstellen

Wenn Sie eine neue Hausverwaltung oder WEG verwalten:

1. Klicken Sie auf **"Neue Organisation"**
2. Geben Sie den **Organisationsnamen** ein (z.B. "WEG Musterstraße 12")
3. Klicken Sie auf **"Erstellen"**

**Ihre Rolle**: Sie sind automatisch **Owner** mit allen Rechten.

### Einer Organisation beitreten

Wenn Sie bereits eingeladen wurden:

1. Öffnen Sie die **Einladungs-E-Mail**
2. Klicken Sie auf den **Einladungslink**
3. Die Einladung ist **24 Stunden gültig**
4. Nach Klick sind Sie automatisch Mitglied

**Ihre Rolle**: Je nach Einladung **Owner** oder **Member**.

---

## 3. Liegenschaft anlegen

Eine Organisation kann mehrere Liegenschaften verwalten.

1. Navigieren Sie zu **"Liegenschaften"**
2. Klicken Sie auf **"Neue Liegenschaft"**
3. Füllen Sie die Daten aus:
   - **Name** (z.B. "Musterstraße 12")
   - **Adresse**
   - **PLZ & Ort**
   - **Baujahr** (optional)
   - **Anzahl Einheiten** (optional)
   - **Gesamtfläche** (optional)
   - **Notizen** (optional)
4. Klicken Sie auf **"Erstellen"**

➡️ **Die Liegenschaft ist jetzt angelegt**

### Einheiten und Eigentümer hinzufügen

1. Öffnen Sie die **Liegenschaft**
2. Wechseln Sie zum Tab **"Einheiten"**
3. Klicken Sie auf **"Neue Einheit"**
4. Geben Sie ein:
   - **Bezeichnung** (z.B. "Wohnung 1.1" oder "Top 3")
   - **MEA (Miteigentumsanteil)** in Bruchform (z.B. "15/100")
5. Klicken Sie auf **"Erstellen"**

**MEA ist wichtig für Abstimmungen!**

### Eigentümer hinzufügen

1. Klicken Sie bei einer Einheit auf **"Eigentümer hinzufügen"**
2. Geben Sie ein:
   - **Vorname & Nachname**
   - **E-Mail** (optional)
   - **Telefon** (optional)
3. Ein Eigentümer kann mehrere Einheiten besitzen

➡️ **Wiederholen Sie dies für alle Einheiten**

---

## 4. Versammlung erstellen

Jetzt können Sie Ihre erste Eigentümerversammlung planen.

1. Navigieren Sie zu **"Versammlungen"**
2. Klicken Sie auf **"Neue Versammlung"**
3. Füllen Sie die Details aus:
   - **Titel** (z.B. "Jahreshauptversammlung 2024")
   - **Datum & Uhrzeit**
   - **Ort/Adresse**
   - **Liegenschaft** auswählen
   - **Beschreibung** (optional)
4. Klicken Sie auf **"Erstellen"**

### Tagesordnung festlegen

1. Klicken Sie auf **"TOP hinzufügen"**
2. Geben Sie ein:
   - **Titel** (z.B. "Begrüßung und Feststellung der Beschlussfähigkeit")
   - **Beschreibung** (optional)
   - **Erfordert Beschluss?** (Ja/Nein)
3. Fügen Sie alle Tagesordnungspunkte hinzu

➡️ **Die Versammlung ist jetzt vorbereitet**

---

## 5. Versammlung durchführen

Am Tag der Versammlung:

### Schritt 1: Versammlungsleitung festlegen

1. Öffnen Sie die **Versammlung**
2. Klicken Sie auf **"Durchführen"**
3. Wählen Sie den **Versammlungsleiter**
4. Wählen Sie den **Protokollführer**
5. Klicken Sie auf **"Weiter"**

### Schritt 2: Teilnehmer erfassen

Das System zeigt alle Eigentümer der Liegenschaft:

1. **Markieren Sie anwesende Eigentümer**
2. Das System berechnet automatisch:
   - Anwesende MEA
   - Beschlussfähigkeit (mindestens 50% erforderlich)
3. Klicken Sie auf **"Weiter"**

**Wichtig**: Nur anwesende Eigentümer können abstimmen!

### Schritt 3: Tagesordnungspunkte abarbeiten

Für jeden TOP:

1. **Ohne Beschluss**: Einfach besprechen und zum nächsten
2. **Mit Beschluss**: Abstimmung durchführen

### Abstimmung durchführen

1. Das System zeigt alle **anwesenden Eigentümer**
2. Für jeden Eigentümer wählen Sie:
   - ✅ **Ja**
   - ❌ **Nein**
   - ⚪ **Enthaltung**
3. Das System berechnet automatisch:
   - **Ja-Stimmen** (nach MEA gewichtet)
   - **Nein-Stimmen** (nach MEA gewichtet)
   - **Enthaltungen** (nach MEA gewichtet)
   - **Ergebnis**: Angenommen oder Abgelehnt (> 50% der anwesenden MEA erforderlich)

**Beispiel:**
- Eigentümer A (MEA 15/100): Ja
- Eigentümer B (MEA 20/100): Ja
- Eigentümer C (MEA 10/100): Nein
- Eigentümer D (MEA 15/100): Enthaltung

**Ergebnis**: 35% Ja, 10% Nein, 15% Enthaltung → **Angenommen** ✅

4. Klicken Sie auf **"Weiter"** für den nächsten TOP

### Schritt 4: Zusammenfassung

1. Das System zeigt alle Beschlüsse
2. Sie sehen das Ergebnis jeder Abstimmung
3. Klicken Sie auf **"Versammlung abschließen"**

➡️ **Die Versammlung ist durchgeführt!**

---

## 6. Protokoll ansehen

Nach der Versammlung:

1. Navigieren Sie zu **"Versammlungen"**
2. Wählen Sie die **abgeschlossene Versammlung**
3. Das Protokoll zeigt:
   - Datum, Ort, Teilnehmer
   - Alle Tagesordnungspunkte
   - Alle Abstimmungsergebnisse
   - Detaillierte Stimmverteilung

Sie können das Protokoll:
- **Ansehen** (im Browser)
- **Als PDF herunterladen** (geplant)
- **Per E-Mail versenden** (geplant)

---

## Zusammenfassung des Workflows

```
1. Anmelden (Google/Microsoft)
   ↓
2. Organisation erstellen
   ↓
3. Liegenschaft anlegen
   ↓
4. Einheiten + Eigentümer hinzufügen
   ↓
5. Versammlung erstellen
   ↓
6. Tagesordnung festlegen
   ↓
7. Versammlung durchführen
   ↓
8. Protokoll ansehen
```

## Nächste Schritte

- **Weitere Mitglieder einladen**: Laden Sie Kollegen ein, die helfen sollen
- **Mehrere Liegenschaften**: Verwalten Sie mehrere Objekte in einer Organisation
- **Tagesordnungsvorlagen**: Erstellen Sie Vorlagen für wiederkehrende TOPs
- **Regelmäßige Versammlungen**: Planen Sie Ihre nächste Versammlung

## Häufige Fragen

### Kann ich eine Versammlung später bearbeiten?

Nein, abgeschlossene Versammlungen können nicht mehr bearbeitet werden. Das Protokoll ist unveränderbar.

### Was passiert bei fehlender Beschlussfähigkeit?

Wenn weniger als 50% der MEA anwesend sind, können Sie trotzdem fortfahren, aber Beschlüsse sind möglicherweise nicht gültig.

### Können Eigentümer remote abstimmen?

Aktuell nicht. Nur anwesende Eigentümer können abstimmen.

### Kann ich das Protokoll nachträglich ändern?

Nein, das Protokoll ist nach Abschluss unveränderbar. Dies gewährleistet die rechtliche Sicherheit.
