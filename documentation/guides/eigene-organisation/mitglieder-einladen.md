# Mitglieder einladen

Fügen Sie weitere Personen zu Ihrer Organisation hinzu, damit diese gemeinsam Versammlungen verwalten können.

## Voraussetzungen

- Sie haben eine Organisation erstellt
- Sie haben die Rolle **Owner**

## Mitglied einladen

1. Navigieren Sie zu **Einstellungen**
2. Wählen Sie den Tab **"Mitglieder"**
3. Klicken Sie auf **"Mitglied einladen"**

## Einladung konfigurieren

### E-Mail-Adresse

Geben Sie die E-Mail-Adresse der Person ein, die Sie einladen möchten.

### Rolle wählen

Wählen Sie eine Rolle für das neue Mitglied:

**Owner**
- Kann Organisation bearbeiten und löschen
- Kann Mitglieder einladen und entfernen
- Kann alle Funktionen nutzen

**Member**
- Kann auf alle Ressourcen zugreifen
- Kann Versammlungen erstellen und durchführen
- Kann keine Organisations-Einstellungen ändern

## Einladung versenden

Klicken Sie auf **"Einladen"**.

Das neue Mitglied erhält eine E-Mail mit einem Einladungslink.

## Einladungs-Gültigkeit

- Die Einladung ist **24 Stunden** gültig
- Der Empfänger kann innerhalb dieser Zeit beitreten
- Danach muss eine neue Einladung versendet werden

## Einladung annehmen

Der Empfänger:
1. Erhält eine E-Mail mit Einladungslink
2. Klickt auf den Link
3. Meldet sich an
4. Wird automatisch zur Organisation hinzugefügt

## Mitglieder verwalten

In der Mitglieder-Übersicht sehen Sie:
- Alle aktuellen Mitglieder
- Deren Rollen (Owner/Member)
- Beitrittsdatum

### Mitglied entfernen

1. Klicken Sie auf **"Entfernen"** beim gewünschten Mitglied
2. Bestätigen Sie die Aktion

Das Mitglied verliert sofort den Zugriff auf die Organisation.

### Rolle ändern

Owner können die Rolle eines Mitglieds von Member zu Owner (und umgekehrt) ändern.

## Häufige Fragen

### Wie viele Mitglieder kann ich einladen?

Es gibt keine Begrenzung für die Anzahl der Mitglieder.

### Kann ich mehrere Owner haben?

Ja, eine Organisation kann mehrere Owner haben. Dies wird empfohlen für den Fall von Abwesenheit.

### Was passiert wenn die Einladung abläuft?

Die Einladung wird ungültig. Sie können einfach eine neue Einladung an dieselbe E-Mail-Adresse senden.

### Kann ein Member zu Owner werden?

Ja, ein Owner kann die Rolle eines Members jederzeit zu Owner ändern.
