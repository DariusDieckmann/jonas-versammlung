# Organisation erstellen

Eine Organisation ist der zentrale Bereich, in dem Sie Ihre Liegenschaften, Versammlungen und Mitglieder verwalten.

## Was ist eine Organisation?

Eine Organisation enthält:
- **Liegenschaften**: Alle verwalteten Gebäude
- **Versammlungen**: Alle geplanten und durchgeführten Versammlungen
- **Mitglieder**: Alle Benutzer mit Zugriff auf die Organisation
- **Tagesordnungsvorlagen**: Wiederverwendbare Vorlagen für Versammlungen

## Schritt 1: Organisation anlegen

Nach dem Login navigieren **Einstellungen**navigieren Sie zu **Organisation**

## Schritt 2: Name eingeben

Geben Sie einen Namen für Ihre Organisation ein:

```
Beispiel: Hausverwaltung Müller GmbH
```
## Schritt 3: Organisation speichern

Klicken Sie auf **"Organisation erstellen"**.

Sie werden automatisch als **Owner** (Eigentümer) der Organisation hinzugefügt.

## Mitglieder hinzufügen

Nach dem Erstellen können Sie weitere Mitglieder zur Organisation einladen:

1. Klicken Sie auf **"Mitglied einladen"**
2. Geben Sie die E-Mail-Adresse ein
3. Wählen Sie die Rolle: **Owner** oder **Member**
4. Klicken Sie auf **"Einladen"**

Der eingeladene Benutzer erhält eine E-Mail mit einem Einladungslink. Die Einladung ist 24 Stunden gültig.

## Rollen und Berechtigungen

| Rolle | Berechtigung |
|-------|--------------|
| **Owner** | Kann Organisation bearbeiten, löschen und Mitglieder verwalten |
| **Member** | Kann auf alle Ressourcen der Organisation zugreifen |

Beide Rollen können:
- Liegenschaften erstellen und verwalten
- Versammlungen planen und durchführen
- Eigentümer und Einheiten verwalten

## Nächste Schritte

- Erstellen Sie eine Liegenschaft in Ihrer Organisation
- Planen Sie Ihre erste Versammlung

## Häufige Fragen

### Kann ich den Namen später ändern?

Ja, Owner können den Organisationsnamen jederzeit in den Einstellungen ändern.

### Kann ich eine Organisation löschen?

Ja, Owner können die Organisation löschen. Beachten Sie, dass dabei alle zugehörigen Daten (Liegenschaften, Versammlungen, etc.) ebenfalls gelöscht werden.

### Was ist der Unterschied zwischen Owner und Member?

Owner können die Organisation verwalten und Mitglieder hinzufügen/entfernen. Members haben Zugriff auf alle Funktionen, können aber keine Organisations-Einstellungen ändern.
