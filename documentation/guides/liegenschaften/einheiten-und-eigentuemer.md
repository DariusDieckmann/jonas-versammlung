# Einheiten und Eigentümer verwalten

Verwalten Sie die Wohneinheiten Ihrer Liegenschaft und ordnen Sie ihnen Eigentümer zu.

## Einheit erstellen

1. Öffnen Sie die Detailansicht einer Liegenschaft
2. Wechseln Sie zum Tab **"Einheiten"**
3. Klicken Sie auf **"Neue Einheit"**

### Einheit-Daten

**Pflichtfelder**
- **Bezeichnung**: z.B. "Wohnung 1.OG links" oder "TOP 3"
- **MEA (Miteigentumsanteile)**: Stimmgewicht (z.B. 125 von 10.000)

**Optionale Felder**
- **Etage**: Stockwerk (z.B. 1, 2, 3)
- **Fläche (m²)**: Wohnfläche in Quadratmetern
- **Notizen**: Zusätzliche Informationen

### Eigentümer direkt hinzufügen

Sie können beim Erstellen der Einheit bereits einen Eigentümer hinzufügen:

**Pflichtfelder**
- **Vorname**: Vorname des Eigentümers
- **Nachname**: Nachname des Eigentümers

**Optionale Felder**
- **E-Mail**: E-Mail-Adresse
- **Telefon**: Telefonnummer  
- **Notizen**: Zusätzliche Informationen

## Eigentümer verwalten

### Eigentümer hinzufügen

1. Öffnen Sie eine Einheit
2. Klicken Sie auf **"Eigentümer hinzufügen"**
3. Geben Sie die Daten ein
4. Speichern Sie den Eigentümer

### Eigentümer bearbeiten

1. Klicken Sie auf den Eigentümer
2. Ändern Sie die Daten
3. Speichern Sie die Änderungen

### Eigentümer entfernen

1. Klicken Sie auf **"Entfernen"** beim Eigentümer
2. Bestätigen Sie die Aktion

## Miteigentumsanteile (MEA)

Die MEA bestimmen das Stimmgewicht bei Abstimmungen in Versammlungen.

**Beispiel:**
- Gesamtanteile: 10.000
- Einheit A: 125 MEA = 1,25% Stimmgewicht
- Einheit B: 250 MEA = 2,5% Stimmgewicht

Bei Abstimmungen wird automatisch nach MEA gewichtet.

## Einheit bearbeiten

1. Öffnen Sie die Liegenschaft
2. Klicken Sie auf die Einheit
3. Klicken Sie auf **"Bearbeiten"**
4. Ändern Sie die Daten
5. Speichern Sie die Änderungen

## Einheit löschen

1. Öffnen Sie die Einheit
2. Klicken Sie auf **"Löschen"**
3. Bestätigen Sie die Aktion

**Achtung:** Dabei werden auch alle zugehörigen Eigentümer gelöscht.

## Häufige Fragen

### Was sind Miteigentumsanteile?

MEA (Miteigentumsanteile) legen fest, welchen Anteil eine Einheit an der Gesamtimmobilie hat. Sie bestimmen das Stimmgewicht bei Beschlüssen.

### Kann eine Einheit mehrere Eigentümer haben?

Ja, Sie können einer Einheit beliebig viele Eigentümer zuordnen. Bei Abstimmungen zählt die Einheit einmal mit ihren MEA.

### Muss ich alle Eigentümer-Kontaktdaten angeben?

Nein, E-Mail und Telefon sind optional. Vorname und Nachname sind Pflicht.

### Wie ändere ich die MEA einer Einheit?

Öffnen Sie die Einheit, klicken Sie auf "Bearbeiten" und ändern Sie die MEA.
