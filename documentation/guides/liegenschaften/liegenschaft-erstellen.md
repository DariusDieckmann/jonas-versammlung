# Liegenschaft erstellen

Eine Liegenschaft repräsentiert ein Gebäude oder eine Immobilie, die Sie verwalten. Für jede Liegenschaft können Sie später Versammlungen planen.

## Voraussetzungen

- Sie haben eine Organisation erstellt
- Sie sind eingeloggt

## Schritt 1: Zur Liegenschaften-Übersicht

1. Navigieren Sie zu **Liegenschaften** im Hauptmenü
2. Klicken Sie auf **"Neue Liegenschaft"**

## Schritt 2: Grunddaten eingeben

### Pflichtfelder

**Name/Bezeichnung**
```
Beispiel: Wohnanlage Musterstraße
```

**Adresse (Straße und Hausnummer)**
```
Beispiel: Musterstraße 42a
```

**PLZ**
```
Beispiel: 12345
```

**Ort**
```
Beispiel: Berlin
```

### Optionale Felder

- **Baujahr**: Jahr der Errichtung (z.B. 2020)
- **Anzahl Einheiten**: Anzahl der Wohnungen/Gewerbeeinheiten
- **Gesamtfläche (m²)**: Gesamtfläche der Liegenschaft
- **Notizen**: Zusätzliche Informationen

## Schritt 3: Liegenschaft speichern

Klicken Sie auf **"Liegenschaft erstellen"**.

Die Liegenschaft wird Ihrer Organisation hinzugefügt und Sie werden zur Detailansicht weitergeleitet.

## Nächste Schritte

Nach dem Erstellen der Liegenschaft können Sie:

1. **Einheiten hinzufügen**: Fügen Sie Wohnungen oder Gewerbeeinheiten hinzu
2. **Eigentümer zuordnen**: Weisen Sie Eigentümer den Einheiten zu
3. **Versammlung planen**: Erstellen Sie eine Versammlung für diese Liegenschaft

## Häufige Fragen

### Kann ich eine Liegenschaft später bearbeiten?

Ja, klicken Sie in der Detailansicht auf "Bearbeiten", um alle Daten zu ändern.

### Muss ich alle Einheiten sofort hinzufügen?

Nein, Sie können Einheiten jederzeit später hinzufügen.

### Kann ich eine Liegenschaft löschen?

Ja, in der Detailansicht können Sie die Liegenschaft löschen. Beachten Sie, dass dabei auch alle zugehörigen Einheiten und Versammlungen gelöscht werden.
